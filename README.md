# OpenBeam X Foundation — Landing Page

This folder contains a minimal static site for **openbeam.org**.

## Files
- `index.html` – one-page landing content
- `styles.css` – minimal styling
- `logo.svg` – simple logo
- `CNAME` – sets custom domain to `openbeam.org` for GitHub Pages
- `README_DEPLOY.md` – step-by-step free deployment guide

## Quick preview
Open `index.html` locally or deploy via GitHub Pages (free).
